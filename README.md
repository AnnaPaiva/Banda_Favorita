## 🧩 Projetos em Destaque

### 🎵 **Website Banda Favorita**
Website profissional desenvolvido para uma banda musical fictícia, com foco em apresentação, agenda de shows, galeria e interação com fãs.

**Funcionalidades:**
- Página institucional completa  
- Agenda dinâmica de eventos  
- Galeria de fotos  
- Formulário de contratação  
- Painel administrativo para gestão de conteúdo  

**Tecnologias:**  
`HTML` · `CSS` · `JavaScript` · `PHP` · `MySQL`

---

### 📇 **CRUD de Contatos — PHP + MySQL**
Sistema completo de gestão de contatos, com autenticação e funcionalidades profissionais.

**Funcionalidades:**
- Login com sessão  
- Adicionar, editar e excluir contatos  
- Busca por nome  
- Ordenação por colunas  
- Layout responsivo  
- Proteção de páginas  

**Tecnologias:**  
`PHP` · `MySQL` · `HTML` · `CSS`

🔗 **Repositório:**  
https://github.com/AnnaPaiva/crud_contatos
